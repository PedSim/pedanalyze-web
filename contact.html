Collaboration & Contributions

PedAnalyze is an open-source project dedicated to advancing pedestrian behavior analysis research. We welcome contributions from researchers, developers, students- anyone interested in this field.

How You Can Contribute

  • Research Collaboration: Share datasets, research findings, or collaborate on new features
  • Code Contributions: Help improve the platform's functionality and user experience
  • Annotations: Annotate YouTube videos to create enough datasets for future work
  • Bug Reports: Report issues or suggest improvements
  • Feature Requests: Propose new features or enhancements
