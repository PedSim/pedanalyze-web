Help & Instructions

  1. Create a Project: On the home page, enter a project name and a YouTube video link, then click "Create".
  2. Open a Project: Click on a project card to start annotating, or load a JSON file for existing annotations.
  3. Annotate: Use the ontology page to tag pedestrian, vehicle, environment, and archetype behaviors frame by frame.
  4. Save/Export: Download your annotations as a JSON file for further analysis.
  5. Need More Help? Contact us via the Contact page for support.
